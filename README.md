# 任务进度看板
作者：郑州大学 ljk；邮箱：lijinke_zzu@zzu.edu.cn

这是一个用于记录任务完成进度的轻量看板，适合施工记录、项目跟踪、学习计划、任务管理等场景。

## 功能

- 添加多个任务
- 为任务添加日期、完成比例和文字标注
- 给任务添加多个标签，按颜色分类查看
- 支持折叠任务和恢复任务
- 支持删除任务、标注和标签
- 支持导出某一天所有任务的完成情况为 Markdown
- 支持导出和导入整份看板备份
- 支持新建空白看板
- 所有数据默认保存在浏览器本地，刷新后不会丢

## 如何打开

1. 把 `index.html`、`app.js`、`styles.css` 放在同一个文件夹里。
2. 用浏览器直接打开 `index.html`。
3. 页面打开后就可以直接开始记录任务。

## 快速上手

### 1. 添加任务

- 在顶部“任务名称”输入框里填写任务名。
- 点击“添加任务”。

### 2. 给任务加进度

- 在任务卡片里填写：
  - 日期
  - 完成比例
  - 文字标注
- 点击“添加标注”。
- 标注会显示在进度条上，也会出现在下方的标注列表中。

### 3. 给任务加标签

- 在任务卡左上方的“输入标签”处输入标签名。
- 按回车，或者点击“添加标签”。
- 同一个任务可以有多个标签。
- 页面顶部会自动出现彩色标签页，点击后可以按标签查看任务。

### 4. 编辑和管理任务

- 点击“编辑名称”修改任务名。
- 点击“折叠任务”把任务收起。
- 点击“恢复任务”把折叠任务拉回进行中。
- 点击“删除任务”删除整条任务。
- 删除和折叠都会弹出确认框，避免误操作。

### 5. 删除标注和标签

- 标注右侧的 `×` 可以删除某条标注。
- 标签右侧的 `×` 可以删除某个标签。

## 导出 Markdown

如果你想记录某一天所有任务的完成情况：

1. 在“导出日期”里选择日期。
2. 点击“导出 Markdown”。
3. 会下载一个 `.md` 文件，里面包含当天所有任务的状态、标签和最新进度。

## 备份和迁移

如果你要在别的电脑上继续使用：

1. 点击“导出看板备份”。
2. 会得到一个 `.json` 备份文件。
3. 到另一台电脑打开同样的网页后，点击“导入看板备份”。
4. 选择这个 `.json` 文件，就能恢复全部任务。

## 新建空白看板

- 点击“新建空白看板”即可清空当前页面的任务，开始一张新的空白板。
- 建议先导出一次看板备份，再新建空白板，避免误清空。

## 保存规则

- 数据默认保存在浏览器本地。
- 刷新页面后，一般不会丢失。
- 如果换浏览器或换电脑，需要用“导出看板备份 / 导入看板备份”进行迁移。

## 推荐使用流程

1. 新建任务。
2. 给任务加标签，例如“施工”“生活”“科研”。
3. 按日期不断补充完成比例。
4. 不再继续的任务折叠起来。
5. 需要汇报时导出 Markdown。
6. 需要迁移时导出备份文件。
